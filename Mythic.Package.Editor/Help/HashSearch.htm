Hash Search

This feature helps you to find out unknown file names in dictionary. There are multiple types of this search:

  • Text search: Tries to match content of the text box to selected entities (either package, block, single or multiple files).
  • Expression search: This is used when you want to match multiple searches at the same time.

    Syntax: text{start-finish}text

    • start - start of the search. Can range from 0 to z.
    • finish - end of the search. Can range from 0 to z.

    Note: Length of start must be the same as length of finish. There can be multiple entries in the search query. Text must be in lower case.

    Warning: This search is time consuming so be patient.

    Examples:

    • data/definitions/hues/hue161{3-7}.bmp
      This query will go through the following file names:

      data/definitions/hues/hue1613.bmp
      data/definitions/hues/hue1614.bmp
      data/definitions/hues/hue1615.bmp
      data/definitions/hues/hue1616.bmp
      data/definitions/hues/hue1617.bmp

    • data/definitions/hues/{aaa-zzz}16{33-77}.bmp
      This query will go through the following file names:

      data/definitions/hues/aaa1633.bmp
      data/definitions/hues/aaa1634.bmp
      data/definitions/hues/aaa1635.bmp
      data/definitions/hues/aaa1636.bmp
      ...
      data/definitions/hues/aaa1677.bmp
      data/definitions/hues/aab1633.bmp
      data/definitions/hues/aab1634.bmp
      ...
      data/definitions/hues/zzz1677.bmp

    • data/definitions/hues/{0000000-zzzzzzz}.bmp
      This query will go through the following file names:

      data/definitions/hues/0000000.bmp
      data/definitions/hues/0000001.bmp
      data/definitions/hues/0000002.bmp
      ...
      data/definitions/hues/0000009.bmp
      data/definitions/hues/000000a.bmp
      data/definitions/hues/000000b.bmp
      ...
      data/definitions/hues/000000z.bmp
      data/definitions/hues/0000010.bmp
      ...
      data/definitions/hues/zzzzzzz.bmp
