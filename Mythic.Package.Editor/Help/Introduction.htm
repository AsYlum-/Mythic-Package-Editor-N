Welcome to use Mythic Package Editor!

Mythic Package Editor is a versatile software product for editing your .uop files. You can:

  • View content of .uop packages.
  • Extract files from .uop packages.
  • Add new files to .uop packages.
  • Delete files inside .uop packages.

We recommend that you read through this manual. It is the easiest way to learn how to use the software's versatile features efficiently.
